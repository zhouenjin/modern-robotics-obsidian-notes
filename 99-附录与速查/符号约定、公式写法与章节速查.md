# 符号约定、公式写法与章节速查

## 1. 公式写法约定

本 vault 统一使用：

- 行内公式：`$...$`
- 独立公式：`$$...$$`

不再使用 `\[...\]`。

## 2. 常见对象速查

| 对象 | 含义 |
| --- | --- |
| $q$ | 广义坐标或构型变量 |
| $\dot{q}$ | 广义速度 |
| $R \in SO(3)$ | 旋转矩阵 |
| $T \in SE(3)$ | 齐次变换矩阵 |
| $\omega$ | 角速度 |
| $V = (\omega, v)$ | twist |
| $F = (\tau, f)$ | wrench |
| $S_i$ | 空间表达的第 $i$ 个关节 screw axis |
| $B_i$ | 本体表达的第 $i$ 个关节 screw axis |
| $M$ | 零位形末端位姿 |

## 4. 最常见误区

> [!warning]
> C-space 不是 workspace。

> [!warning]
> $SO(3)$ 是姿态空间，$SE(3)$ 是位姿空间。

> [!warning]
> twist 的下半部分不只是“平移方向”，它还编码了轴相对原点的位置关系。

> [!warning]
> PoE 的空间表达和本体表达不是两种不同的运动学，而是同一个几何问题的两种参考表达。

> [!warning]
> Jacobian 不是只会“求导”的矩阵，它在课程里同时连接：
> - 关节速度和末端 twist；
> - 末端 wrench 和关节力矩；
> - 奇异性和 manipulability。

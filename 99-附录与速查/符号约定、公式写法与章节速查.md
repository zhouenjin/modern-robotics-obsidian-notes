## 2. 常见对象速查

| 对象 | 含义 |
| --- | --- |
| $q$ | 广义坐标 / 构型变量 |
| $\dot{q}$ | 广义速度 |
| $R \in SO(3)$ | 旋转矩阵 |
| $T \in SE(3)$ | 齐次变换矩阵 |
| $\omega$ | 角速度 |
| $V = (\omega, v)$ | twist |
| $F = (\tau, f)$ | wrench |
| $S_i$ | space screw axis |
| $B_i$ | body screw axis |
| $M$ | 零位时末端位姿 |
| $T_{sb}$ | `{b}` 相对 `{s}` 的位姿，并在 `{s}` 中表达 |
| $\operatorname{Ad}_T$ | twist / wrench 的 6x6 变换矩阵 |
| $J_s, J_b$ | space / body Jacobian |

## 4. 最常见误区

> [!warning]
> C-space 不是 workspace。

> [!warning]
> $SO(3)$ 只表示姿态，$SE(3)$ 才表示位姿。

> [!warning]
> twist 下半部分的 $v$ 不是“某个点的真实速度”，而是 twist 的线速度部分。

> [!warning]
> PoE 的 space 表达和 body 表达不是两种不同运动学，而是同一问题的两种参考系表达。

> [!warning]
> 对 $T_{ab}$，第二个下标告诉你“谁相对谁”，第一个下标告诉你“结果在哪个坐标系里表达”。

> [!warning]
> 若 screw axis 在 space frame 中表达，则左乘：
> $T' = e^{[S]\theta}T$
> 若 screw axis 在 body frame 中表达，则右乘：
> $T' = Te^{[B]\theta}$

> [!warning]
> twist 按 $\operatorname{Ad}_T$ 变换，wrench 按其对偶形式变换：
> $V_a = \operatorname{Ad}_{T_{ab}}V_b$
> $F_b = \operatorname{Ad}_{T_{ab}}^T F_a$

> [!warning]
> 逆运动学通常不是唯一解，也不保证存在解。

> [!warning]
> 数值逆运动学依赖初值，不一定收敛到你想要的那组解。

> [!warning]
> 闭链机构里常见的是“逆运动学相对容易、正运动学相对困难”，不要照搬开链直觉。

> [!warning]
> 闭链奇异点不只是“Jacobian 不可逆”这么简单，还可能涉及约束 Jacobian 降秩。
